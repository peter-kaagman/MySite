# MySite – Perl CMS met Dancer2 en DBIx::Class

MySite is een zelfgebouwd CMS gebaseerd op Perl en Dancer2, met DBIx::Class als database layer.

Doel van dit project is niet alleen een werkende site, maar vooral het begrijpen van:
- opbouw van webapplicaties
- scheiding tussen content, data en presentatie
- observability en gedrag van applicaties

De site draait op mysite.prjv.nl en bevat artikelen over k3s, cloudplatformen en automatisering.

## Motivatie

Na jaren WordPress gebruikt te hebben, wilde ik beter begrijpen hoe een CMS intern werkt.
Daarom is gekozen voor een eigen implementatie met Dancer2 en SQLite.

DBIx::Class speelt hierin een centrale rol als data abstraction layer.

## Status

Work in progress. Focus ligt op functionaliteit en inzicht, niet op polish.
